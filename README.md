# Daily Briefing AI Chatbot Issues

A daily briefing service that captures LinkedIn comments and discussions regarding public-facing AI agents or chatbots misbehaving or hallucinating.

## Overview

This project monitors LinkedIn (via search/social listening techniques) to identify instances where users are discussing AI chatbot failures. These discussions are then processed, summarized by an AI, and sent as a daily email briefing.

## Features

- **LinkedIn Monitoring**: Targeted search for chatbot hallucination and misbehavior reports.
- **AI Summarization**: Intelligent filtering and categorization of findings.
- **Daily Email**: A concise report delivered to your inbox every day.
